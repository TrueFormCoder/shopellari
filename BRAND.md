## Quick reference (do not re-derive — see canon for full spec)

| Layer | Lock |
|---|---|
| Headings + product names | **EB Garamond** (semibold, 500–700) |
| Body + UI | **Inter** (300–600) |
| Metadata + IDs | **DM Mono** (400–500) |
| Ecosystem anchor color | `#534AB7` (Aetherion) |
| Canvas | `#fdfaf6` (warm cream) |
| Brand accents | Read from `src/data/brands.ts` |
| New brands | PR to `src/data/brands.ts` only |
| CTAs | Status-driven: `Buy Now` / `Subscribe` / `Join Waitlist` / `Contact` |

## Out of scope for this repo

- Cormorant Garamond — sealed for MirrorProtocol products only
- Cream/black/gold MirrorProtocol palette — sealed for MirrorProtocol products only
- Release gates, readiness scores, license intake forms — deferred until ≥10 paying customers per product

## Rule

Brand book prose does not establish brand registry. New brand families enter via PR to `src/data/brands.ts`. Names that appear in strategy docs without a corresponding registry entry are treated as candidate **products**, not brands.
